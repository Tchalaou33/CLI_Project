# CLI Project

Gives details on Studio Gihbli films
Includes all of Studio Gihbli's productions for the past 34 years

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'CLI Project'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install CLI_Project

## Usage

$ film-finder
